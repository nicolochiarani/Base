# Decimal to Base-N Converter

## About the Project
This is a simple C++ program that converts numbers from a decimal base to a user-defined base between 2 and 62 (using numbers, uppercase, and lowercase letters). 

It is designed to be a learning tool to explain two main concepts:
* How **recursion** works in programming.
* The mathematics behind converting a number from a decimal base to base *n*.

I am a beginner eager to learn as much as possible, and this project is part of my learning journey. Thank you for being here and checking out my code!

## Features
* **Custom Base Conversion:** Supports any base from 2 to 62.
* **Input Handling:** The program is built to safely handle all user inputs, both valid and invalid, preventing crashes.

## Environment
* **Language:** C++
* **OS:** Created and tested on Linux (Ubuntu).
